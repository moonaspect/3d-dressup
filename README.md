# 3D Dress-Up Game

Nintendo 3DS-style anime girl dress-up game built in Unity (C#).

## Features
- Hair, eyes, lips, blush, skin gradient, clothes, shoes, backgrounds
- Save/Load character system
- Randomize character button
- Character rotation & blinking animation
- Export screenshots

## Tools
- Unity
- Blender (3D modeling)
- Krita (textures)

For detailed design & architecture, see `GDD.pdf`.
